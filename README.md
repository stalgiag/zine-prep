# Zine Prep

A browser-based tool for preparing PDFs for zine printing. All processing happens locally in your browser - nothing is uploaded and no data is collected.

## Formats

- **Booklet** - Saddle-stitch imposition for fold-and-staple booklets
- **Un-impose** - Reverse an imposed PDF back to reading order
- **Mini Zine** - Classic 8-page single-sheet zine with one cut
- **Half-Fold** - Simple 4-page bi-fold card
- **Quarter Size** - Pocket-size saddle-stitch booklet (4-up)
- **2-Up Proof** - Side-by-side pages for proofing spreads
- **Accordion** - Z-fold continuous strip layout
